## Platform Components

<Steps>
  <Step title="Data Collection">
    Crawler scripts fetch Movies, TV Series, and People data from TMDB API with intelligent ID tracking
  </Step>
  <Step title="Stream Ingestion">
    Kafka producers simulate real-world traffic with mixed record types (new, change, old)
  </Step>
  <Step title="Stream Processing">
    Spark Structured Streaming consumes from Kafka with lightweight parsing and DLQ for fault tolerance
  </Step>
  <Step title="Batch Refinement">
    Airflow orchestrates Bronze → Silver → Gold transformations with deduplication and change detection
  </Step>
  <Step title="Multi-Model Sync">
    Refined data syncs to ClickHouse (analytics), Neo4j (graph), and Pinecone (vector search)
  </Step>
</Steps>

## Tech Stack

The platform leverages industry-leading technologies:

- **Languages**: Python, SQL
- **Data Ingestion**: Apache Kafka, Apache Spark
- **Orchestration**: Apache Airflow
- **Storage**: Delta Lake, MinIO (S3-compatible object storage)
- **Databases**: ClickHouse (OLAP), Neo4j (Graph), Pinecone (Vector)
- **Infrastructure**: Docker, Kubernetes

<Note>
  This platform implements a Medallion Architecture that bridges the gap between raw Kafka streams and specialized downstream applications, ensuring data reliability through ACID transactions and optimizing performance with intelligent change-tracking.
</Note>
