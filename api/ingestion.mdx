## Overview

The Ingestion service is responsible for loading entertainment data from local files, enriching records with metadata, and publishing them to Kafka topics for downstream processing.

## Core Components

### Main Entry Point

<ParamField path="main" type="function">
  Main entry point for the ingestion service that orchestrates the entire data loading pipeline.

  **Location**: `src/ingestion/main.py:12`

  **Process Flow**:
  1. Sets up logging configuration
  2. Loads application settings
  3. Initializes Kafka producer
  4. Iterates through data files (movies, TV series, persons)
  5. Enriches records with metadata
  6. Sends records to Kafka topics
  7. Flushes buffer periodically
</ParamField>

## Usage Examples

### Running the Ingestion Service

```python
from ingestion.main import main

# Start the ingestion service
main()
```

### Loading Configuration

```python
from ingestion.config.settings import load_settings
import os

# Set environment
os.environ['APP_ENV'] = 'production'

# Load settings
settings = load_settings()
print(f"Kafka server: {settings.kafka.server.bootstrap_servers}")
print(f"Movie topic: {settings.kafka.topics.movie}")
```

### Using the File Loader

```python
from pathlib import Path
from ingestion.loader.file_loader import iter_full

# Iterate through all data files
data_root = Path("/path/to/data")
for record, data_type, data_label in iter_full(data_root):
    print(f"Processing {data_type} ({data_label}): {record.get('id')}")
```

### Enriching Records

```python
from ingestion.preprocessor.record_processor import enrich_record

original_record = {
    "movie_id": 12345,
    "title": "Example Movie",
    "overview": "A great film"
}

enriched = enrich_record(
    record=original_record,
    data_type="movie",
    data_label="new"
)

print(enriched)
# Output includes: data_type, data_label, timestamp
```

### Using Kafka Producer

```python
from ingestion.producer.kafka_producer import KafkaProducerService
from ingestion.config.settings import load_settings

settings = load_settings()
producer = KafkaProducerService(settings)

# Send records
for i in range(100):
    record = {"id": i, "name": f"Item {i}"}
    producer.send(settings.kafka.topics.movie, record)
    
    # Flush every 50 records
    if i % 50 == 0:
        producer.flush()

# Close producer
producer.close()
```

## Configuration Example

```yaml
kafka:

```

## Data Flow

<Steps>
  <Step title="Load Files">
    `iter_full()` reads JSONL files in round-robin fashion from data folders
  </Step>
  
  <Step title="Enrich Records">
    `enrich_record()` adds metadata (type, label, timestamp) to each record
  </Step>
  
  <Step title="Send to Kafka">
    `KafkaProducerService.send()` publishes records to appropriate topics
  </Step>
  
  <Step title="Buffer Management">
    Records are buffered and flushed periodically based on `max_buffer` setting
  </Step>
</Steps>

## Error Handling

<Warning>
The ingestion service logs errors but continues processing when:
- Individual records fail to parse
- Kafka buffer is full (with backpressure)
- Network issues occur (with retries)

Fatal errors will stop the service and log exception details.
</Warning>

## Performance Considerations

<Tip>
- **Buffer Size**: Adjust `max_buffer` based on record size and throughput
- **Flush Interval**: 1-second delay between flushes prevents overwhelming Kafka
- **Parallelism**: Currently single-threaded; consider parallel processing for larger datasets
- **Memory**: Round-robin iteration minimizes memory usage
</Tip>
